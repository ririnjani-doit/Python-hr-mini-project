# Candidate Selection System

## Overview

This project simulates an HR candidate screening process using Python.

The program calculates weighted candidate scores based on:

- English
- Python
- Work Experience

Then it classifies each candidate into recommendation categories.

---

## Skills Demonstrated

- Python
- List
- Dictionary
- Function
- If Else
- Loop
